Minesweeper
===========
Version: 1.0

Thomas Shaffer <tjshaffer21@gmail.com>

Description:
------------
Minesweeper clone.

Controls:
---------
Left Mouse Button  - Reveal selected point.
Right Mouse Button - Flag selected point.
R - Restart game
Q - Quit game

Requirements:
-------------
ncurses
c++0x capable compiler
